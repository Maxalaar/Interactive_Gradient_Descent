# Interactive Gradient Descent

---

## 🌐 Run Live Web Demo

🔗 Live demo: [https://interactive-gradient-descent.onrender.com/](https://interactive-gradient-descent.onrender.com/)

An interactive 3D visualization tool to understand **gradient descent optimization** on complex loss landscapes.

* 3D loss surface visualization
* Click-to-start optimization paths
* Compare optimizers
* Multiple benchmark loss functions

---

## 🚀 Run Locally (recommended for better performance)

For a smoother experience (faster rendering, higher responsiveness), it is recommended to run the project locally instead of the hosted version.

### 1. Clone the repository

```bash
git clone https://github.com/your-repo/interactive-gradient-descent.git
cd interactive-gradient-descent
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
python Interactive_Gradient_Descent/main.py
```

### 4. Open in browser

```
http://127.0.0.1:8050
```
